ETC Solutions — Component Library

Design System

A living showcase of the ETC Solutions design system — typography tokens with fluid sizing, and the core components built from them. Every element on this page references the shared Flowbite token set.

Foundations

Typography

Two families — Lora (serif) for headings & quotes, Google Sans for everything else. Editorial styles are kept separate from the raw size scale; sizes ≥ 24px scale fluidly with the token value as the ceiling.
